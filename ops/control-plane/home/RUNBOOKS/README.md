# Runbooks

This folder holds repeatable procedures Codex can use to operate the workspace.

## Available Runbooks
- `daily-operator.md`: default daily operating loop for business execution
- `3dvr-production.md`: default checks and actions for the live 3DVR stack
- `3dvr-market-icp.md`: paid-market definition, fit criteria, and offer mapping for 3DVR
- `3dvr-segment-angles.md`: default landing-page and pitch angles for the first three paid wedges
- `3dvr-interview-outreach.md`: research script, outreach copy, and CRM tag set for market validation
- `warm-lead-member-journey.md`: outreach journey from warm lead to active 3DVR member
- `warm-lead-crm-internal.md`: internal CRM structure, statuses, scoring, and daily workflow
- `warm-lead-crm-template.csv`: local CSV CRM template
- `warm-lead-crm.jsonl`: local JSONL CRM seed for automation and scripting
