3dvr Pocket Workstation

Turn your phone into a builder console.

Pocket Workstation gives you one portable place for notes, commands, projects, and guided next steps. It works in the browser right now and can bootstrap a real Termux setup on Android.

Copy the install route Open the dashboard

Install in Termux

Run this once on Android inside Termux:

curl -Ls https://3dvr.tech/install | bash

Then run 3dvr connect and link the short code in the Pocket Workstation dashboard.

What it does

Simple first. Useful immediately.

  • Save notes for deployments, ideas, and live project context.
  • Save reusable commands so you do not keep rebuilding your memory from scratch.
  • Track projects and the next action that moves them forward.
  • Use the lightweight helper to turn “I need to deploy this” into concrete steps.

Why it matters

One system across devices.

  • Android gets local power through Termux.
  • iPhone gets the same dashboard as a clean control panel.
  • Desktop gets the same workspace with more room.
  • The center stays your projects, not a generic terminal toy.

How it works

Three clear steps

1. Install Run the one-line install command in Termux.
2. Connect Run 3dvr connect, then approve the short code in the dashboard.
3. Pull your setup Use 3dvr commands pull or 3dvr projects to bring your workspace in.
