3dvr local agent

Install the cockpit on a new device.

This installs the 3dvr command line agent, links the 3dvr command, creates local config, and gets the device ready for portal OAuth.

One command

curl -Ls https://3dvr.tech/install-agent | bash

Works on Debian, Termux, WSL, and macOS when Git, Node, npm, and curl or wget are available.

1. Install

Run the command above from a terminal on the device you want to use.

2. Verify

Run 3dvr doctor to confirm Node, config, and the CLI link.

3. Connect

Run 3dvr connect, approve OAuth in the portal, then import the connection.

Manual fallback

git clone https://github.com/tmsteph/3dvr-agent.git
cd 3dvr-agent
./install.sh
3dvr setup
3dvr doctor

The installer stores the checkout under ~/.3dvr/agent by default. Override with THREEDVR_HOME, THREEDVR_AGENT_DIR, or THREEDVR_BIN_DIR.
